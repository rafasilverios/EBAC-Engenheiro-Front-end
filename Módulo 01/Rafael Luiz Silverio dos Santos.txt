O HTML é uma linguagem de marcação de hipertexto e é utilizado na estrutura de uma página web. Com ele podemos dividir a página em cabeçalho, corpo e rodapé e, criar menus, formatar textos, incluir imagens na página, criar links para outras páginas, dentre outros. O HTML tem sua importância para a criaçao de páginas devido sua fácil utilização, compatibilidade, acessibilidade, transmissão e integração. 

O CSS por sua vez, é quem dá o estilo para a página web. Ele permite que o criador do site aplique em um documento de HTML, diversos tipos de formações, através de regras. O CSS é importante para alinhar, editar, definir espaçamento, remover e formatar objetos, textos e imagens. Com ele podemos deixar o site com uma aparência muito mais agradável para o usuário.

O Javascript é o unico dentre os três que é de fato uma linguagem de programaçao. É importante, pois com ele são implementados os itens de maior complexidade nos sites, são criadas interaçoes avançadas com o usuário e é feita toda a comunicação do site com o back-end. Se com o HTML o site é estruturado e com o CSS o estilo do site é criado, com o JavaScript realizamos toda a programação que interage tanto com o usuário (Client Side), quanto com os servidores (Server Side).
